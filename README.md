# Fibonacci em C (Análise de Complexidade)

Projeto desenvolvido em linguagem C com o objetivo de implementar diferentes abordagens para o cálculo da sequência de Fibonacci, analisando suas complexidades computacionais.

## Funcionalidades
- Implementação recursiva (complexidade exponencial O(2^n))
- Implementação iterativa (complexidade linear O(n))
- Implementação otimizada (complexidade O(log n))

##  Conceitos aplicados
- Lógica de programação
- Recursividade
- Estruturas de repetição
- Análise de complexidade de algoritmos

## Tecnologias
- Linguagem C

##  Objetivo
Comparar diferentes formas de resolver o mesmo problema e entender o impacto da complexidade no desempenho.
